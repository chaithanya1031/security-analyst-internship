# Task 2: Basic Firewall Configuration with UFW

## Objective
To configure a basic firewall using UFW that allows SSH traffic and blocks HTTP traffic.

## Tool Used
- UFW (Uncomplicated Firewall)

## Configuration Steps
- Reset firewall rules
- Deny all incoming traffic by default
- Allow outgoing traffic
- Allow SSH (port 22)
- Deny HTTP (port 80)

## Security Significance
- Firewalls protect systems from unauthorized access
- Restricting open ports reduces attack surface
- UFW provides simple and effective firewall management
