# Task 7: Vulnerability Scanning with Nikto

## Objective
To identify potential vulnerabilities in a web server using Nikto.

## Tool Used
- Nikto

## Target
- Local Apache web server (http://localhost)

## Findings
Nikto identified multiple security issues including:
- Server version disclosure
- Missing security headers
- Enabled HTTP methods
- Apache status information exposure

## Security Impact
These issues can help attackers perform reconnaissance and client-side attacks.

## Conclusion
Nikto is an effective vulnerability scanning tool that helps security analysts quickly identify web server misconfigurations.
